KAPITEL 1 · THEORIE

Exponentielles Wachstum und Abnahme

VERGLEICH

Lineares vs. exponentielles Wachstum

Beim linearen Wachstum wird in jedem Schritt derselbe Betrag addiert.
Beim exponentiellen Wachstum wird in jedem Schritt mit demselben Faktor multipliziert.

EigenschaftLinearExponentiell
Was ist konstant?DifferenzenQuotienten
Beispiel (Start 100)100 → 105 → 110 → 115100 → 105 → 110,25 → 115,76
Wie prüfen?Differenz aufeinanderfolgender WerteQuotient aufeinanderfolgender Werte
GraphGeradeKurve (steigend/fallend)
ERKENNUNGSREGEL

Wie erkenne ich exponentielles Wachstum?

Bilde die Quotienten aufeinanderfolgender Werte:

B(1)/B(0) , B(2)/B(1) , B(3)/B(2) , ...

Sind alle Quotienten gleich (= konstant), liegt exponentielles Wachstum vor.

Beim linearen Wachstum sind die Differenzen konstant.

KERNFORMEL

Die Grundformel des exponentiellen Wachstums

B(n) = B(0) · qⁿ
  • B(n) – Bestand nach n Zeitschritten
  • B(0) – Anfangsbestand (Startwert)
  • q – Wachstumsfaktor
  • n – Anzahl der Zeitschritte
BEISPIEL

Bevölkerungswachstum

Eine Stadt mit 50 000 Einwohnern wächst jährlich um 2 %.

1
Wachstumsfaktor bestimmen
q = 1 + 2/100 = 1,02
2
Gleichung aufstellen
B(n) = 50 000 · 1,02ⁿ
3
Nach 10 Jahren
B(10) = 50 000 · 1,02¹⁰ ≈ 60 950
4
Nach 25 Jahren
B(25) = 50 000 · 1,02²⁵ ≈ 82 030
ZUSAMMENHANG

Wachstumsfaktor q und Wachstumsrate p

q = 1 + p/100

Wachstum (q > 1):

  • Zunahme um p % → q = 1 + p/100
  • z.B. +5 % → q = 1,05
  • Verdopplung → q = 2

Abnahme (0 < q < 1):

  • Abnahme um p % → q = 1 − p/100
  • z.B. −3 % → q = 0,97
  • Halbierung → q = 0,5
BEISPIEL – WACHSTUMSFAKTOR BESTIMMEN

Guthaben wächst von 5000 € auf 6077,53 € in 4 Jahren

1
Grundformel ansetzen
5000 · q⁴ = 6077,53
2
Dividieren
q⁴ = 6077,53 / 5000 = 1,2155
3
4. Wurzel ziehen
q = ⁴√1,2155 ≈ 1,05
4
Wachstumsrate
p = (1,05 − 1) · 100 = 5 %

Wertverfall eines Autos

Ein Neuwagen kostet 25 000 €. Sein Wert sinkt jährlich um 12 %.

Jahr (n)0123456

TYPISCHE FEHLER

Achtung!

  • Abnahme um 15 % bedeutet q = 0,85 — nicht q = −0,15 und nicht q = 1,15
  • Die Einheit von n beachten: Jahre, Monate oder Tage?

KAPITEL 2 · THEORIE

Exponentialfunktionen

DEFINITION

Die allgemeine Exponentialfunktion

f(x) = a · bˣ
  • a ≠ 0 – der Anfangswert (y-Achsenabschnitt), denn f(0) = a · b⁰ = a
  • b > 0, b ≠ 1 – die Basis (Wachstums- bzw. Zerfallsfaktor)
  • b > 1 – Graph steigt (exponentielles Wachstum)
  • 0 < b < 1 – Graph fällt (exponentielle Abnahme)
GRAPHENVERGLEICH
EIGENSCHAFTEN

Wichtige Eigenschaften von f(x) = a · bˣ (a > 0)

  • Für x kann jede beliebige Zahl eingesetzt werden (auch negative und Dezimalzahlen)
  • Die Funktionswerte sind immer positiv – der Graph liegt immer oberhalb der x-Achse
  • Der Graph berührt die x-Achse nie, er nähert sich ihr nur immer weiter an
  • Der y-Achsenabschnitt ist immer der Punkt (0 | a)
  • Es gibt keine Nullstelle (der Graph schneidet die x-Achse nicht)
WACHSTUM

b > 1 → steigend

Beispiele: f(x) = 2ˣ, g(x) = 3ˣ, h(x) = 1,5ˣ

y-Achse: Punkt (0 | a)
ABNAHME

0 < b < 1 → fallend

Beispiele: g(x) = 0,5ˣ, k(x) = 0,8ˣ

Halbwertszeit berechenbar
VERDOPPLUNGSZEIT & HALBWERTSZEIT

Definition

Verdopplungszeit:

Zeitspanne, in der sich der Bestand verdoppelt.

qⁿ = 2 → n = log_q(2)

Halbwertszeit:

Zeitspanne, in der sich der Bestand halbiert.

qⁿ = 0,5 → n = log_q(0,5)

f(x) = 100 · 1,1ˣ (Wachstum um 10 % pro Schritt)

VORGEHEN

Exponentialfunktion durch zwei Punkte aufstellen

  • Beide Punkte in f(x) = a · bˣ einsetzen → zwei Gleichungen
  • Gleichungen durcheinander dividieren → a fällt weg
  • b berechnen (Wurzel ziehen)
  • b einsetzen → a berechnen
  • Probe mit dem anderen Punkt

KAPITEL 3 · THEORIE

Logarithmen & Exponentialgleichungen

DEFINITION

Was ist ein Logarithmus?

Der Logarithmus ist die Umkehrung des Potenzierens.

bʸ = x ⟺ y = log_b(x)

Eselsbrücke: „log_b(x): b hoch was ergibt x?"

UMRECHNUNG
PotenzschreibweiseLogarithmusschreibweise
2³ = 8log₂(8) = 3
10² = 100log₁₀(100) = 2
5³ = 125log₅(125) = 3
3⁴ = 81log₃(81) = 4
b⁰ = 1log_b(1) = 0 (immer!)
KOPFRECHNEN ÜBEN

Logarithmen im Kopf berechnen

  • log₂(16) = ? → 2 hoch was ergibt 16? → 2⁴=16 → 4
  • log₃(27) = ? → 3 hoch was ergibt 27? → 3³=27 → 3
  • log₁₀(1000) = ? → 10³=1000 → 3
  • log₅(1) = ? → 5⁰=1 → 0
TASCHENRECHNER

Logarithmen mit dem TR berechnen

Wenn auf dem Taschenrechner nur log(x) ohne Basis steht, ist damit der Zehnerlogarithmus log₁₀(x) gemeint.

Umrechnungsformel (falls kein log_b-Taste vorhanden):

log_b(x) = log(x) / log(b)

Beispiel: log₃(50) = log(50) / log(3) = 1,699 / 0,477 ≈ 3,561

PROBE

Probe für log₃(50) ≈ 3,561: Gilt 3^3,561 ≈ 50? → Ja! ✓

EXPONENTIALGLEICHUNGEN LÖSEN

So geht's: bˣ = c lösen

bˣ = c ⟹ x = log_b(c)

Vorgehen bei komplexeren Gleichungen

1
Vorfaktor wegdividieren (falls vorhanden)
500 · 1,08ˣ = 1200 | ÷ 500 → 1,08ˣ = 2,4
2
Logarithmus anwenden
x = log₁,₀₈(2,4)
3
Mit Taschenrechner
x = log(2,4)/log(1,08) ≈ 11,38

Wachstumsprozesse modellieren

VORGEHEN

Von Daten zur Modellfunktion

1
Daten analysieren

Differenzen und Quotienten aufeinanderfolgender Werte bilden.

2
Wachstumsart bestimmen
Konstante Differenzen → lineares Modell: f(x) = mx + b
Konstante Quotienten → exponentielles Modell: f(x) = a · bˣ
3
Gleichung aufstellen

Parameter a und b (bzw. m und b) bestimmen.

4
Modell überprüfen

Stimmen die berechneten Werte mit den Daten überein?

5
Prognosen erstellen

Gleichung für Vorhersagen nutzen.
