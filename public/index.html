KAPITEL 1 · THEORIE

Exponentielles Wachstum und Abnahme

VERGLEICH

Lineares vs. exponentielles Wachstum

Beim linearen Wachstum wird in jedem Schritt derselbe Betrag addiert.
Beim exponentiellen Wachstum wird in jedem Schritt mit demselben Faktor multipliziert.

EigenschaftLinearExponentiell
Was ist konstant?DifferenzenQuotienten
Beispiel (Start 100)100 → 105 → 110 → 115100 → 105 → 110,25 → 115,76
Wie prüfen?Differenz aufeinanderfolgender WerteQuotient aufeinanderfolgender Werte
GraphGeradeKurve (steigend/fallend)
ERKENNUNGSREGEL

Wie erkenne ich exponentielles Wachstum?

Bilde die Quotienten aufeinanderfolgender Werte:

B(1)/B(0) , B(2)/B(1) , B(3)/B(2) , ...

Sind alle Quotienten gleich (= konstant), liegt exponentielles Wachstum vor.

KERNFORMEL

Die Grundformel des exponentiellen Wachstums

B(n) = B(0) · qⁿ
  • B(n) – Bestand nach n Zeitschritten
  • B(0) – Anfangsbestand (Startwert)
  • q – Wachstumsfaktor
  • n – Anzahl der Zeitschritte
BEISPIEL

Bevölkerungswachstum

Eine Stadt mit 50 000 Einwohnern wächst jährlich um 2 %.

1
Wachstumsfaktor bestimmen
q = 1 + 2/100 = 1,02
2
Gleichung aufstellen
B(n) = 50 000 · 1,02ⁿ
3
Nach 10 Jahren
B(10) = 50 000 · 1,02¹⁰ ≈ 60 950
ZUSAMMENHANG

Wachstumsfaktor q und Wachstumsrate p

q = 1 + p/100

Wachstum (q > 1):

  • Zunahme um p % → q = 1 + p/100
  • z.B. +5 % → q = 1,05

Abnahme (0 < q < 1):

  • Abnahme um p % → q = 1 − p/100
  • z.B. −3 % → q = 0,97
BEISPIEL – WACHSTUMSFAKTOR BESTIMMEN

Guthaben wächst von 5000 € auf 6077,53 € in 4 Jahren

1
Grundformel ansetzen
5000 · q⁴ = 6077,53
2
Dividieren
q⁴ = 6077,53 / 5000 = 1,2155
3
4. Wurzel ziehen
q = ⁴√1,2155 ≈ 1,05
BEISPIEL – EXPONENTIELLE ABNAHME

Wertverfall eines Autos

Ein Neuwagen kostet 25 000 €. Sein Wert sinkt jährlich um 12 %.

B(n) = 25 000 · 0,88ⁿ
TYPISCHE FEHLER

Achtung!

  • Abnahme um 15 % bedeutet q = 0,85 — nicht q = −0,15
  • Die Einheit von n beachten: Jahre, Monate oder Tage?

KAPITEL 2 · THEORIE

Exponentialfunktionen

DEFINITION

Die allgemeine Exponentialfunktion

f(x) = a · bˣ
  • a ≠ 0 – der Anfangswert (y-Achsenabschnitt)
  • b > 0, b ≠ 1 – die Basis
  • b > 1 – Graph steigt
  • 0 < b < 1 – Graph fällt
GRAPHENVERGLEICH
EIGENSCHAFTEN

Wichtige Eigenschaften von f(x) = a · bˣ (a > 0)

  • Die Funktionswerte sind immer positiv
  • Der Graph berührt die x-Achse nie
  • Der y-Achsenabschnitt ist immer (0 | a)
  • Es gibt keine Nullstelle
VERDOPPLUNGSZEIT & HALBWERTSZEIT

Definition

Verdopplungszeit:

qⁿ = 2 → n = log_q(2)

Halbwertszeit:

qⁿ = 0,5 → n = log_q(0,5)
VORGEHEN

Exponentialfunktion durch zwei Punkte aufstellen

  • Beide Punkte einsetzen → zwei Gleichungen
  • Gleichungen dividieren → a fällt weg
  • b berechnen, dann a
BEISPIEL

P(1 | 6) und Q(4 | 48)

1
Gleichungssystem
(I) a · b¹ = 6    (II) a · b⁴ = 48
2
Dividieren (II):(I)
b³ = 48/6 = 8 → b = 2
3
Ergebnis
f(x) = 3 · 2ˣ

KAPITEL 3 · THEORIE

Logarithmen & Exponentialgleichungen

DEFINITION

Was ist ein Logarithmus?

bʸ = x ⟺ y = log_b(x)

Eselsbrücke: „b hoch was ergibt x?"

UMRECHNUNG
PotenzschreibweiseLogarithmusschreibweise
2³ = 8log₂(8) = 3
10² = 100log₁₀(100) = 2
b⁰ = 1log_b(1) = 0 (immer!)
TASCHENRECHNER

Basiswechselformel

log_b(x) = log(x) / log(b)

Beispiel: log₃(50) = log(50) / log(3) ≈ 3,561

EXPONENTIALGLEICHUNGEN

bˣ = c lösen

bˣ = c ⟹ x = log_b(c)
1
Vorfaktor wegdividieren
500 · 1,08ˣ = 1200 | ÷ 500 → 1,08ˣ = 2,4
2
Logarithmus anwenden
x = log(2,4)/log(1,08) ≈ 11,38
